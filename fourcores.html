Core4 Tools

The four core tools are...

  • Loops
  • Conditionals
  • Variables
  • Functions

Loops

  • Technical Definition: A sequence of commands that continue until it matches the conditon.
  • Real Life Analogy: It is like a sushi bar on a conveyor belt, it keeps going until it is told to stop.

Conditionals

  • Technical Definition: Actions or computations that happen if the condition is true or false.
  • Real Life Analogy: If it is raining, then I will use an umbrella.

Variables

  • Technical Definition: A value that is assigned to a name.
  • Real Life Analogy: Everytime x has 10 added to it, that is called y. So y = x+10.

Functions

  • Technical Definition: A routine, it can be used to make code more efficient.
  • Real Life Analogy: A soda machine which you input money and you output a soda.
